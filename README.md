Anywhere 随启随用的静态文件服务器
========

Running static file server anywhere. 随时随地将你的当前目录变成一个静态文件服务器的根目录。

## Installation
```
npm install anywhere -g
```

## Execution
```
anywhere
// or with port
anywhere 8000
// or with port and does not automatically open the home page
anywhere 8000 false
```
## Visit
```
http://localhost:8080
```
执行命令后，默认浏览器将为您自动打开主页。
